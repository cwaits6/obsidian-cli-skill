# Keyboard Shortcuts

Essential Obsidian keyboard shortcuts for fast navigation.

## Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd+O` | Quick open |
| `Cmd+P` | Command palette |
| `Cmd+Shift+F` | Search vault |
| `Cmd+E` | Toggle edit/preview |
| `Cmd+Click` | Open link in new tab |

## Editing

| Shortcut | Action |
|----------|--------|
| `Cmd+B` | Bold |
| `Cmd+I` | Italic |
| `Cmd+K` | Insert link |
| `Cmd+]` | Indent |
| `Cmd+[` | Outdent |

## Custom

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+D` | Open daily note |
| `Cmd+T` | Insert template |
